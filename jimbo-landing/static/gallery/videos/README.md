# Gallery Videos

This folder contains all videos for the Jimbo Landing Page gallery.

## Structure
- Place videos in this folder
- Supported formats: `.mp4`, `.webm`, `.mov`
- Recommended size: 1920x1080 (Full HD)
- Optimize videos for web (compressed, web-ready formats)

## File Naming Convention
Use descriptive names:
- `project-demo-2024.mp4`
- `interview-tech-conference.mp4`
- `behind-scenes-development.mp4`

## Adding Videos
1. Add video files to this folder
2. Update the gallery data in `/routes/galeria.tsx`
3. Add video descriptions and metadata
4. Consider adding thumbnail images

Videos added here will be automatically available at:
`/gallery/videos/filename.mp4`
