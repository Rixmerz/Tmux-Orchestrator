# Gallery Photos

This folder contains all photos for the Jimbo Landing Page gallery.

## Structure
- Place photos in this folder
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
- Recommended size: 1200x800px or higher
- Optimize images for web before uploading

## File Naming Convention
Use descriptive names:
- `project-name-01.jpg`
- `event-conference-2024.jpg`
- `behind-scenes-studio.jpg`

## Adding Photos
1. Add photo files to this folder
2. Update the gallery data in `/routes/galeria.tsx`
3. Add photo descriptions and metadata

Photos added here will be automatically available at:
`/gallery/photos/filename.jpg`
